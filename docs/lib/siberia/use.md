## use
Подключает `модули` в __глобальный контекст__.<br>
Не поддерживает макросы.<br>
__Не рекомендуется__ к использованию.

### Применение

1. `(use module0 moduleN)`<br>
`module0` `moduleN` - _модули_.

### Применение

```pht
((use pht)
(app-fn
    (println "Здравствуй,")
    (println "Товарищ!")))
```
